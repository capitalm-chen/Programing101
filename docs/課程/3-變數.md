# javascript 中的變數

```javascript
var a = 1;
```

# 原始型別變數

## 數字

```javascript
var a = 100;
var b = -200;
var c = 3.1415926;
```

## 布林值

```javascript
var a = true;
var b = false;
```

# 複雜型別變數

## 字串

```javascript
var a = "The quick brown fox jumps over the \"lazy\" dog.";
document.write( a );
```

## 陣列

```javascript
var a = [1, 1, 2, 3, 5, 8];
document.write( a[3] );
```

## 物件

```javascript
var a = {"first": 1,
    "second": 2,
    "third": 3};
document.write( a["second"] );
document.write( a.third );
```
